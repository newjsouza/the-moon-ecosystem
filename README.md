# 🌙 The Moon — Middleware Cognitivo Universal

Ecossistema de IA pessoal orientado a agentes autônomos.
Projetado para operar com **Custo Zero** e **mínima intervenção humana**.

## Arquitetura
- **Orchestrator** — núcleo central de roteamento e lifecycle
- **NexusIntelligence** — mente de convergência cross-domain
- **SemanticMemoryWeaver** — grafo de conhecimento local
- **BettingAnalystAgent** — análise esportiva (APEX + Kelly)
- **EconomicSentinel** — inteligência financeira multi-módulo
- **SkillAlchemist** — descoberta e integração autônoma de ferramentas
- **AutonomousDevOpsRefactor** — qualidade de código contínua
- **OmniChannelStrategist** — distribuição de conteúdo multicanal
- **HardwareSynergyBridge** — integração nativa com Zorin OS
- **WatchdogAgent** — compliance de custo zero e saúde do sistema
- **MoonBot** — interface unificada via Telegram

## Stack
Python 3.10+ | AsyncIO | Groq Cloud | FastAPI | GTK3 | PipeWire | D-Bus

## Filosofia
> "Não é um conglomerado de scripts. Es um ecossistema."
